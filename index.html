Project Saphire

Project Saphire is a work in progress web application. It allows users of any device (Android, Windows PC, IOS, MacOS) to connect to and manage their FileServer. Saphire is as a web based file browser similar to onedrive and googledrive. Users can manage, create, upload and download files. Saphire was mainly designed to work with Samba File shares.
